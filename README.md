# 🌿 GreenReach · 그린리치

> **위치정보 기반 도시 녹지 접근성 시민 플랫폼**

"내 주변 공원이 얼마나 가까운지, 데이터로 보여드립니다"

---

## 🚀 빠른 시작

### 프론트엔드 실행

```bash
cd frontend
npm install
npm run dev
```

브라우저에서 **http://localhost:5173** 접속

### 백엔드 실행 (선택)

```bash
cd backend
pip install -r requirements.txt
python main.py
```

API 문서: **http://localhost:8000/docs**

| 페이지 | 경로 | 설명 |

### 프론트엔드
- **React 19** + TypeScript + Vite
- **Tailwind CSS** — 스타일링

- 25개 자치구 녹지 통계
- AI 취약지 예측 데이터 (12개 동)

## 📊 접근성 점수 계산 방식

```
총점 (0~100) = 거리 점수 (0~50) + 밀도 점수 (0~30) + 면적 점수 (0~20)
```

| 등급 | 점수 | 의미 |
|------|------|------|
| A | 80~100 | 도보 5분 내 공원 접근 가능 |
| B | 65~79 | 도보 10분 내 공원 접근 가능 |
| C | 50~64 | 도보 15분 내 공원 접근 가능 |
| D | 35~49 | 공원까지 상당한 거리 소요 |
| F | 0~34 | 녹지 접근성 개선 시급 |

> 직선거리 × 1.3 (서울 평균 도로 네트워크 계수) = 실제 도보 거리 추정

## 🎯 비즈니스 모델

시민에게는 **무료**로 제공하고, 데이터를 필요로 하는 기업·기관에서 수익을 창출합니다.

| 단계 | 수익원 | 타겟 | 비고 |
|------|--------|------|------|
| ① 핵심 | **B2B API 제휴** | 직방·다방·호갱노노 등 부동산 플랫폼 | 녹지 접근성 점수 위젯 API 제공 |
| ② 연계 | **건설사 마케팅 자료** | 신축 분양 건설사·시행사 | "공원까지 도보 O분" 분양 자료 활용 |
| ③ 장기 | **공공 지원사업** | 환경부·국토부 데이터 바우처 | 탄소중립·15분 도시 정책 연계 |
| ④ 확장 | **지자체 대시보드** | 구청 도시계획과 | 트랙레코드 확보 후 B2G 영업 |
